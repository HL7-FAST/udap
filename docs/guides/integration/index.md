# Integration Tutorials

Learn how to integrate the FAST Security Reference Implementation with various FHIR servers and technology stacks.

!!! tip "Prerequisites"
    Before starting these tutorials, ensure you have:
    
    - A running instance of the FAST Security server (see [Getting Started](../../setup/getting-started.md))
    - A valid certificate for JWT signing (see [Certificate Generation](../../features/certificate-generation.md))
    - Basic understanding of UDAP workflows
